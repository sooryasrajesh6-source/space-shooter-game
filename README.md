# 🚀 Space Shooter Game

An exciting Space Shooter game built using HTML, CSS, and JavaScript.

## Features
- Keyboard and mobile controls
- Enemy spaceships
- Shooting system
- Score tracking
- Lives system
- Explosion effects
- Responsive design
- Game Over and Restart functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript
